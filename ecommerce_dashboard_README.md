# E-Commerce Sales Dashboard

An interactive sales analytics dashboard built with vanilla JavaScript and Chart.js, featuring real-time filtering and data visualization.

![Dashboard Preview](dashboard_preview.png)

## 🎯 Live Demo
[View Dashboard](https://cphulpagar-byte.github.io/ecommerce-sales-dashboard/)

## 📊 Key Features

- **4 KPI Cards** — Total Revenue, Orders, Avg Order Value, Conversion Rate
- **Interactive Filters** — Filter by Time Period, Category, and Region
- **Revenue Trend Chart** — Monthly revenue visualization with smooth animations
- **Category Distribution** — Doughnut chart showing sales by product category
- **Top Products Table** — Best performing items with performance indicators
- **Regional Performance** — Bar visualization of revenue by geographic region
- **Orders Trend** — Monthly order volume analysis
